CDP / CA Spectrum Notes
  • Each link populates CISCO-CDP-MIB cdpCacheTable on both Cisco devices (routers & switches only — Windows servers do not run CDP).
  • OIDs pushed in real-time to running agents; links persist across restarts.
  • CA Spectrum discovers neighbours via cdpCacheDeviceId, cdpCacheDevicePort, and cdpCacheAddress.
  • For root-cause analysis, ensure CA Spectrum is set to use CDP topology discovery on the community public.
